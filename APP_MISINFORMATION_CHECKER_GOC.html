Misinformation-resistant data checker

Check whether a public data claim has enough source, context, metadata and limitation information before it is shared, cited or used in a briefing.

Challenge 10 · Bilingual standalone prototype · Local deterministic scoring

Important: This prototype does not determine legal truth or falsity. It provides diagnostic feedback on evidence quality and misinformation-risk signals.

Challenge summary

What the app does

It helps users check whether a statistic or data claim includes the basic evidence needed for responsible verification.

🔎 Source checker 🧭 Context builder 🧾 Evidence reporter

Public value

The checker strengthens data literacy by making source, reference period, geography, unit, method and limitations visible.

Priority score

86

Strong challenge fit: public trust, transparent data use, responsible communication and source verification.

Claim checker

Paste a public claim or public URL, then identify what evidence is visible. Validation happens when you submit the form.

There is a problem

    Use a non-identifying alias. Do not enter a real name or personal information.

    Choose a domain or use auto-detect from the claim text.

    Example: Did Canada's inflation rate hit 10% in 2025?

    Optional. Add a public source link, table ID or citation if the claim provides one.

    Evidence visible in the claim

    Select all evidence elements that appear in the claim or source.

    Potential misinformation-risk signals

    Select all risk signals that may apply. The result will show why these lower confidence.

      Privacy, accessibility and responsible use

      Privacy

      The app uses aliases only and stores practice attempts in localStorage. It does not send claim text to a server.

      Accessibility

      The prototype uses semantic HTML, labelled fields, keyboard-accessible controls, visible focus states, error summaries, data tables and print styles.

      Responsible use

      The checker supports learning and evidence review. It does not produce final decisions, legal conclusions or official Government of Canada verification.
