National Data Literacy Sandbox Suite

A DS Challenge 2026 prototype that helps learners practise statistical thinking, source evaluation, accessible visualization and geographic interpretation.

Standalone HTML prototype · no server storage · demonstration data only unless a source link is opened

Before you start

This page is a training prototype. Demonstration values are synthetic and are used to teach interpretation, not to report official statistics. Use the linked Statistics Canada sources for authoritative data.

Sandbox tools

Choose a tool. All tools run in the browser using demonstration data and local rule-based logic.

Local statistical assistant

Ask about rounding, sampling error, coefficient of variation, data suppression, metadata or chart interpretation. The assistant is deterministic and does not send content to an external AI service.

Example: Why does random rounding matter?

Bilingual statistical dictionary

Select a concept to see a plain-language definition, a caution and a source path for further learning.

Interactive matrix explorer

Select a demonstration dataset. Use the matrix to practise comparing indicators and identifying data-quality cautions.

Accessible data visualizer

Choose a demonstration dataset and chart type. The chart is paired with a table and text summary.

Geographic community profile simulator

Select a simulated census tract or use a demonstration preset. The map has a text alternative and all tract buttons are keyboard-accessible.

DS Challenge 2026 project proposal

Problem

Many learners can open data tables but struggle to judge quality, metadata, geography, uncertainty and visualization choices. This slows data literacy and weakens evidence-based analysis.

Solution

Create a bilingual sandbox where users practise statistical reasoning through an assistant, dictionary, data matrix, accessible visualizer and geographic profile simulator.

Public value

The prototype supports students, analysts and public servants by turning statistical methods into task-based learning activities with source links and accessibility safeguards.
