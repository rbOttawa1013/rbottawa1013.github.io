National Data Literacy Sandbox Suite

Practise statistical thinking, source evaluation, accessible visualization and geographic interpretation in one bilingual prototype.

Live public-data attempt · automatic demonstration fallback · no server storage

Before you start

The sandbox first tries to load public Statistics Canada Web Data Service data in your browser. If the public API, network, or browser policy blocks the request, the page falls back to clearly labelled demonstration data.
