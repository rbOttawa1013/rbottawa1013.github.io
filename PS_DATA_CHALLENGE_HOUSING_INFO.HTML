Housing Supply and Affordability Assistant

Enter a Canadian postal code to create a cautious housing information profile using public data sources, local/national comparisons, trend visuals and a constrained explanation assistant.

Alpha prototype · Public data only · No personal information required

Before you start

Postal codes are approximate location keys. Demonstration indicators compare local teaching values against a national baseline and must be verified before operational use.

      Privacy, accessibility and responsible use

      Privacy

      The form asks for postal code only. It does not ask for names, addresses, income, contact information or household details.

      Accessibility

      The prototype uses semantic sections, labelled controls, visible focus, an error summary, keyboard-operable buttons, chart data tables, screen-reader trend text and print styles.

      Responsible AI

      The explanation assistant uses local rules. It does not call an external AI service and does not make decisions about people or eligibility.

      Required departmental validation
      • Official branding and Canada.ca implementation review.
      • Official languages review for English/French parity.
      • Accessibility testing with keyboard, screen reader, zoom, mobile and print scenarios.
      • Privacy, security, data quality, legal and policy review before production use.
