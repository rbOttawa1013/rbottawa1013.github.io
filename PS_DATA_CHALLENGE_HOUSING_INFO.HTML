Housing Supply and Affordability Assistant

Enter a Canadian postal code to create a cautious housing information profile using public source links, approximate geography, demonstration indicators, local and baseline comparisons, and trend visuals.

Alpha prototype · Public data links · Demonstration values · No personal information required

Before you start

Postal codes are approximate location keys. Indicator values in this prototype are demonstration values for learning and design validation. They are not official local housing statistics and must not be used for decisions.

Find housing information by postal code

Use a postal code only. Do not enter a street address, name, income, phone number, email address or other personal information.

        Privacy, accessibility and responsible use

        Privacy

        The form asks for postal code only. The postal code, approximate coordinates and generated profile may be stored in this browser until reset. JSON exports include postal code and approximate coordinates.

        External services

        The prototype may call OpenNorth, Open Government Canada, OpenStreetMap and Canada.ca search. Do not enter personal information in the postal-code field or search box.

        Accessibility

        The prototype uses semantic sections, labelled controls, visible focus, an error summary, keyboard-operable buttons, chart data tables, map text alternatives, screen-reader trend text and print styles.

        Responsible AI

        The explanation assistant uses local rules. It does not call an external AI service and does not make decisions about people, eligibility, credit, rent, property value or neighbourhood quality.

        Required departmental validation
        • Official branding and Canada.ca implementation review.
        • Official languages review for English/French parity.
        • Accessibility testing with keyboard, screen reader, zoom, mobile and print scenarios.
        • Privacy, security, data quality, legal and policy review before production use.
