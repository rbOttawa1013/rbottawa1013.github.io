Canadian flood readiness dashboard

Enter a Canadian postal code or, where public geocoding is available, a street address to view a map-centred flood-readiness profile with live hydrometric stations, weather context, flood-map references, risk scoring and emergency preparedness guidance.

Alpha prototype · Not an official Government of Canada production service · Date modified: 2026-06-25

Before you start

This tool supports awareness and planning. It does not determine whether a property is inside a regulated flood plain, and it does not replace emergency alerts or local evacuation instructions.

Interpretation rule

Visible water, low elevation or nearby flood layers do not prove legal flood-plain status. Use this dashboard to identify which authoritative source to check next.

Sources, APIs and limitations

The page uses public services and defensive fallbacks. If a public map service blocks browser access, is unavailable, or has a changed layer name, the dashboard reports that condition rather than silently presenting mock data as authoritative.

Reality check

There is no single open national API that provides legal floodplain status, flood prediction and Alert Ready messages for every address. This static app combines live hydrometric data, weather layers, federal flood-map references, source links and an explainable client-side risk indicator.

Accessibility, privacy and data-quality gates

  • Privacy: postal-code lookup does not store the value. Street-address lookup is optional and sent only to the public geocoder to resolve coordinates.
  • Accessibility: the page includes skip navigation, visible focus states, semantic landmarks, labelled controls, an error summary, text alternatives for charts and print support.
  • Print: the map is no longer hidden in print output. Use landscape orientation and enable background graphics if the browser suppresses tile colours.
  • Data quality: the page distinguishes live API data, reference layers, unavailable layers and non-authoritative guidance.
  • Security: no API keys, tokens or credentials are embedded.
