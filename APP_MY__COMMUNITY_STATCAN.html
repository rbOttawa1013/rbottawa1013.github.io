My Canada, My Community

Enter a postal code to find the best local statistical match and review model cards for place, population, change, housing, mobility and inclusion.

Postal code only. No external AI or trackers.

Before you start

The app matches the postal code to the closest available local profile: exact FSA profile when available, then city or province fallback. Prototype values must be validated against official Statistics Canada geography.
