My Canada, My Community

Enter a postal code to find a local statistical best match and review model cards for location, demography, population, housing, mobility and language.

Postal code only. No full address is collected. No external AI or trackers are used.

Before you start

The app matches a postal code to a public geography where available, then uses a local or regional demonstration profile when official local table values are not loaded. Matching mechanics, caveats and source questions are kept inside the expandable commentary panels so the main cards remain easy to read.
