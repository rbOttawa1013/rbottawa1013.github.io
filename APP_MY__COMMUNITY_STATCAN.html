My Canada, My Community

Enter a postal code to create a first-pass community profile for location, demography, population, housing, mobility and language.

No external AI is used. Postal-code lookup may call the public OpenNorth Represent API. The map may load OpenStreetMap content. Saved comparisons are stored only in this browser using local storage.

Before you start

The app matches a postal code to a public geography where available, then builds a community profile. Postal-code areas do not always match Census boundaries.
