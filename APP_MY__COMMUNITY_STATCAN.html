My Canada, My Community

Explore a Canadian community from a postal code using live geographic lookup, real Statistics Canada population data, accessible charts, and neutral research prompts.

Alpha prototype. Designed to align with Government of Canada web, accessibility, privacy and official languages expectations.

Before you start

The tool uses a postal code to find an approximate public geographic profile. It does not collect names, street addresses, emails or account information. Live population values come from Statistics Canada Table 17-10-0009-01 when the Web Data Service is available.

Accessibility, privacy and responsible-use gates

Privacy

  • The form asks for a postal code only.
  • No names, street addresses, emails, accounts, analytics or trackers are used.
  • Exported JSON is generated locally in the browser.

Accessibility

  • Skip links, landmarks, one H1, visible focus and labelled form controls are included.
  • Charts include equivalent tables and plain-language summaries.
  • The page supports keyboard use, reflow and print output.

Responsible data use

  • Fallback values are labelled as demonstration data.
  • Prompts avoid partisan, religious and sensitive personal-attribute conclusions.
  • Users are told to verify source data before operational use.
