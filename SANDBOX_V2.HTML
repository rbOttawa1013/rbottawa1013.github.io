This is a training prototype. Demonstration values are synthetic and are used to teach interpretation, not to report official statistics.

        • Synthetic demonstration values are used for learning and testing.

        • Charts include equivalent tables so information is not available only visually.
        • Any future production version must use verified source data and departmental validation.
