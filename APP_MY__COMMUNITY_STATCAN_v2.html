My Canada, My Community

A DS Challenge 2026 prototype for exploring a Canadian community through public data source links, accessible charts, maps and research prompts.

Alpha prototype · Not an official Government of Canada production service

Before you start

This prototype preloads a demonstration profile so results are available immediately. Displayed indicators are demonstration estimates unless marked otherwise. Use the linked Statistics Canada sources for authoritative figures.

Find a community profile

Enter a Canadian postal code to resolve a local geography and generate a demonstration learning profile.

Privacy notice for this prototype

Use a postal code only. The prototype sends the postal code to the public OpenNorth Represent API to resolve a location. It does not ask for a name, full street address, account, email address or other personal information. It does not store data on a server.

Example: K1A 0A6 or M5V 2N4. Do not enter a full street address.
