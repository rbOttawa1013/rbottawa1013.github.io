    About this prototype

    Privacy and responsible use

    This prototype stores mock drafts and scores in this browser only. It does not use external AI, analytics, authentication or a backend.

    API concept panel

    This prototype does not call the live Statistics Canada API by default. A production version may use Web Data Service methods such as getCubeMetadata, getDataFromVectorsAndLatestNPeriods and getFullTableDownloadCSV after review.

    Accessibility

    The page uses semantic regions, one H1, keyboard-operable controls, visible focus, labelled fields, error summaries, accessible tables and print styles. Human testing is still required.

    Date modified: 2026-06-26
