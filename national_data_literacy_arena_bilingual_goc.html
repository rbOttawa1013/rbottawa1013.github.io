Public Service Data Challenge

Build practical data skills using real public data, structured challenges, and transparent rubrics.

Static bilingual prototype for GitHub Pages or local browser demonstration.

How this prototype works

  1. Choose a data challenge.
  2. Inspect the dataset metadata and quality notes.
  3. Draft a mock response using an alias only.
  4. Score the response using a deterministic rubric.
  5. Generate a printable evidence report.

    About this prototype

    Privacy and responsible use

    This prototype stores mock drafts, selected challenge, selected dataset and rubric scores in this browser only. It does not use external AI, analytics, authentication or a backend.

    API concept panel

    This prototype does not call the live Statistics Canada API by default. A production version may use Web Data Service methods after review, with fallback behaviour and source-quality checks.

    Accessibility

    The page uses semantic regions, one H1, keyboard-operable controls, visible focus, labelled fields, visible required text, error summaries, accessible tables, chart data tables and print styles. Human testing remains required.

    Requires validation

    Official branding, content, accessibility, privacy, security, official languages and legal/policy review are required before any production use.
