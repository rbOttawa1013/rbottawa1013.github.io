National Data Literacy Challenge Arena

Practise reading larger Statistics Canada-style tables, answer questions, and save local scores by player or team.

Prototype status: This page uses simplified training extracts and validated Statistics Canada source links. Player names and scores stay in this browser. Use a non-personal alias.

Introduction and instructions

This learning prototype helps users practise data interpretation, source verification and responsible use of official statistics. It is built as a lightweight static web app for the DS Challenge 2026 context.

Each module contains a larger data table, a source link, a subject-matter graphic and randomized multiple-choice questions with explanations.

1. Review sources

Use the dataset links below and in each module to verify the official table, frequency and definitions.

2. Manage players

Add a player or team alias and set the active player before starting a module.

3. Read the table

Open a module, inspect rows, columns, units and notes, then begin the quiz.

4. Check the leaderboard

Review best scores, attempts, badges and total player score in the local leaderboard.

      DS Challenge 2026 project proposal summary

      Problem

      Many users are expected to work with official data but have limited practical training in interpreting tables, checking sources and avoiding common analytical mistakes.

      Solution

      A reusable learning arena that turns source-linked data interpretation into short, accessible, bilingual-ready challenge modules.

      Public value

      The prototype builds confidence, encourages source verification, supports privacy-by-design and can scale to additional learning pathways.

      MVP status

      The current file is a working static HTML application with local player management, source-linked modules, randomized quizzes, feedback, badges and local leaderboard storage.
