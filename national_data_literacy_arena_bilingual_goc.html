Data Literary Challenge

Choose one of 10 public-data categories, review a source, and answer custom statistical-literacy questions about methods, parameters, concepts and trends.

Privacy: use an alias only. Scores stay in this browser. Live mode calls public Statistics Canada endpoints only.

How it works

  1. Choose a dataset category.
  2. Use demo values or try a live public source check.
  3. Answer 10 custom multiple-choice questions.
  4. Review competency feedback and medals.
  5. Write an optional 3-sentence briefing.
