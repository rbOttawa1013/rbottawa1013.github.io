Dataset briefing challenge

Choose one of 10 challenge categories, review a public dataset link and complete a 10-question statistical briefing quiz.

Privacy: use an alias only. Scores stay in this browser. Live mode calls public Statistics Canada endpoints only.

How it works

  1. Choose a dataset category.
  2. Use demo data or try a live public source check.
  3. Answer 10 statistical literacy questions.
  4. Review competency feedback and earn bronze, silver or gold.
  5. Write an optional 3-sentence briefing.

    About

    Privacy

    Alias, answers, briefing notes, language, data mode and scores stay in localStorage. The prototype uses no external AI, analytics, authentication or backend submission.

    Accessibility

    The quiz uses semantic sections, one H1, keyboard controls, fieldsets, legends, an error summary, text feedback and scoped table headers.

    Demo and live data

    Demo values are fixed teaching examples. Live mode attempts public Statistics Canada source/API calls from the browser; if the request fails, the app remains usable in demo mode.
