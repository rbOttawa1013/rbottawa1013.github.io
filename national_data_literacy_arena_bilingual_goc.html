Data Literary Challenge

Privacy: no personal information is required. Quiz scoring stays in this browser session. Refresh data opens public Statistics Canada tables.

How it works

  1. Choose a module.
  2. Select Refresh data to open the public Statistics Canada table.
  3. Select Start quiz for a custom 10-question questionnaire about that dataset.
