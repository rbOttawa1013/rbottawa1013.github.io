Data Literary Challenge

Privacy: No personal information is required. Quiz scoring stays in this browser. Data source URLs open public Statistics Canada tables.

How it works

  1. Choose a module.
  2. Use the dataset table to open the public Statistics Canada source URL.
  3. Select Start quiz for a custom 10-question questionnaire about that dataset.
  4. Use the Leaderboard tab to track score by module and print progress.
