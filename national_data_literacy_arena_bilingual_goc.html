Data Literary Challenge

Privacy: use an alias only. No personal information is required. Refresh data opens public Statistics Canada source links.

How it works

  1. Choose a dataset category.
  2. Select Refresh data to open the public Statistics Canada source table.
  3. Use the custom quiz guide to ask dataset-specific questions on methods, parameters, concepts and trends.

Custom quiz guide

Use these ten question types for each dataset. Replace generic yes-no wording with dataset-specific wording.

  1. What exact Statistics Canada table should be cited?
  2. What frequency and reference period should be stated?
  3. What geography can the dataset support?
  4. What unit, denominator, index or rate concept matters?
  5. What must be held constant before comparing values?
  6. What trend can be described without overclaiming?
  7. What chart type fits the dataset structure?
  8. What limitation must be disclosed?
  9. What individual-level or causal claim would overstate the data?
  10. What sentence separates evidence, interpretation, limitation and next step?
