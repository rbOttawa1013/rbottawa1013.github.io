National Data Literacy Challenge Arena

Practice reading larger Statistics Canada-style tables, answer ten questions per module, and save local scores by player.

Prototype status: This page uses simplified training extracts and validated Statistics Canada source links. Player names and scores stay in this browser. Do not enter personal information.

Introduction and instructions

Each module now contains a larger data table with multiple rows and columns, a validated Statistics Canada table link, improved subject-matter graphics, and ten data-literacy questions. The embedded data are simplified training extracts; use the linked Statistics Canada tables for official values and definitions.

1. Review sources

Use the dataset links below and in each module to verify the official table, frequency and definitions.

2. Manage players

Add a player or team and set the active player before starting a module.

3. Read the table

Open a module, inspect the larger rows and columns, then begin the quiz.

4. Check the leaderboard

Review best score, attempts and total player score in the local scoreboard.
